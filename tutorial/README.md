## 5) React

### Installation

```bash
// to install react
npm install --save react react-dom

// to work with webpack
npm install --save babel-loader babel-preset-es2015 babel-preset-react
```

Update .babelrc

```JavaScript
{
  "presets": ["es2015", "react"]
}
```
