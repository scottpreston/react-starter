# React Challenge / Lab

Do the following steps to test your React Knowledge.

1. Open your ES6/Webpack Challenge.
2. Create an App Component, and a User Component.
3. Populate your user component with the following data so that it shows a list.
    http://scottpreston.github.io/html/data.json
4. Add a form with 2 fields, name and email.
5. Update the list created in #3 with the information from your form.
